2. Какие данные обрабатываются

2.1. Данные, которые предоставляет Родитель при регистрации и использовании Приложения:

  • номер мобильного телефона Родителя (используется для входа по коду из SMS и связи по вопросам сервиса);
  • имя Родителя и/или имя ребёнка (используется для персонализации заданий, например обращения к ребёнку по имени);
  • обращения в поддержку (например, по электронной почте).

2.2. Данные, собираемые автоматически: технические сведения об устройстве (модель, версия операционной системы), IP-адрес, идентификаторы для доставки push-уведомлений, обезличенная статистика прохождения заданий (используется для улучшения контента и работы Приложения).

2.3. Данные о геолокации Приложением не запрашиваются и не обрабатываются.

2.4. Специальные категории персональных данных (сведения о здоровье, биометрические данные и т. п.) не обрабатываются.

2.5. Фотографии, документы, удостоверяющие личность ребёнка, и иные избыточные для целей Приложения данные Оператором не запрашиваются; Родителю не следует указывать такие данные в обращениях в поддержку.

3. Цели обработки

  • регистрация и вход в Приложение;
  • предоставление развивающих заданий и сохранение прогресса их выполнения ребёнком;
  • направление уведомлений о новых заданиях, достижениях и событиях сервиса;
  • оформление и продление платной подписки, приём оплаты;
  • поддержка пользователей;
  • обеспечение безопасности, предотвращение злоупотреблений, улучшение работы Приложения;
  • исполнение требований законодательства.

Данные обрабатываются на основании согласия Родителя, в целях заключения и исполнения пользовательского соглашения, а также для исполнения требований законодательства Российской Федерации.

5.1. Оператор не продаёт персональные данные и не передаёт их третьим лицам в маркетинговых целях.

5.2. Приложение не является социальной сетью: имя, номер телефона и данные о прогрессе ребёнка не отображаются другим пользователям Приложения и третьим лицам.

5.3. Для работы отдельных функций привлекаются сервисы-обработчики, которым передаётся только необходимый для соответствующей функции объём данных: доставка push-уведомлений (Google Firebase, Apple); отправка SMS с кодом входа; хостинг и техническая поддержка инфраструктуры Приложения; приём платежей за подписку — платёжный сервис ЮKassa (ООО НКО «ЮМани»), которому передаются данные, необходимые для проведения платежа (номер телефона, сумма и идентификатор подписки). Реквизиты банковской карты вводятся непосредственно в форме ЮKassa и Оператору не передаются.

5.4. Данные могут быть переданы уполномоченным государственным органам при наличии законных оснований.

5.5. Основные данные хранятся в Российской Федерации. При доставке push-уведомлений часть технических данных (идентификатор устройства) может обрабатываться на серверах за пределами РФ; такая передача осуществляется с соблюдением требований законодательства о персональных данных.
